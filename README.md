# sound-keyboard

## Pin wiring guide

| GPIO Pin | Function    | Notes                                                   |
|----------|-------------|---------------------------------------------------------|
| 2        | Status LED  | Active high. Change if your board uses a different pin. |
| 4        | Encoder CLK | -                                                       |
| 5        | Encoder DT  | -                                                       |
| 6        | Skip Back   | Wire between pin and GND.                               |
| 7        | Skip Ahead  | Wire between pin and GND.                               |
| 8        | Mute        | Wire between pin and GND.                               |
| 9        | Pause       | Wire between pin and GND.                               |
